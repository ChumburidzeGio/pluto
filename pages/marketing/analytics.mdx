# Analytics

Before you start sending traffic to your landing page make sure you have analytics in place. The standard is adding Google Analytics, you can also add Facebook Pixel and Linkedin Insight Tag depending on where you gonna run your ad campaign.

[https://amplitude.com/](https://www.mikavas.com/p/da7d54f9-0b54-4aca-b3ce-b810142280b1/Amplitude) or [https://mixpanel.com/](https://www.mikavas.com/p/da7d54f9-0b54-4aca-b3ce-b810142280b1/Mixpanel) for user behaviour analysis

I would recommend installing session recording software so you can watch exactly what people do on the website and how they read/interact with information you put on landing page. Both of these services have free version so you can use for validation and later as well for free up to some point.

**Session Recordings:** HotJar/FullStory/[Inspect](https://www.inspectlet.com/)
